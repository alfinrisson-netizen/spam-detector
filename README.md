# spam-Email-detector

This is a machine learning project that detects spam emails that uses NLP and Naive Bayes.

## Features

- Text classification (spam/ not spam)
- Built using CountVectorizer + Naive Bayes
- Simple Streamlit web app
- Real-time prediction

## Tech Stack

- Python
- Scikit-learn
- Pandas
- Streamlit

## How to run

1. Install dependencies
2. Train model
3. Run app


## Model

- Algorithm: Multinomial Naive Bayes
- Vectorizer: CountVectorizer

## Deployment

Can be deployed using Streamlit Cloud or github pages.
